# Instructions

This document explains how OpenCode agents read and use instruction files in this repository.

Structure

- docs/instructions.md — this overview.
- docs/example-instruction.md — a short, runnable instruction example.

Usage

- Store clear, goal-oriented instructions in docs/ or .instructions/ for the agent to consume.
- Keep instructions short; specify inputs, expected outputs, and any constraints.

Example

- See docs/example-instruction.md for a minimal template.
